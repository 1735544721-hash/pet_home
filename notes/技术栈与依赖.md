# 技术栈与依赖

## 后端
- Java 17（`pom.xml`）
- Spring Boot 3.2.0 / Spring Web / Spring Security
- MyBatis-Plus 3.5.7
- MySQL Connector/J
- Knife4j OpenAPI3
- Hutool 5.8.25
- JWT（`java-jwt` 4.4.0）
- Spring Boot Mail / AOP
- Lombok

## 运行与配置要点
- 服务端口：`server.port=12345`
- 数据源：MySQL（`pet_home`）
- Swagger UI：`/swagger-ui.html`
- Knife4j Basic：`admin/admin`
- 前端代理：`/api` -> `http://localhost:12345`
