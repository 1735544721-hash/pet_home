# 功能模块清单

## 用户与权限
- 用户登录/资料/账号维护（`UserController`）
- 角色与菜单权限管理（`RoleController` / `MenuController`）

## 宠物与健康
- 宠物信息与分类维护（`PetController` / `PetCategoryController`）
- 健康记录与疫苗接种管理（`PetHealthRecordController` / `PetVaccinationController`）

## 领养与寄养
- 领养申请管理（`AdoptionController`）
- 寄养信息管理（`BoardingController`）

## 商品与订单
- 商品与购物车管理（`ProductController` / `CartController`）
- 订单创建/支付/售后管理（`OrderController` / `OrderReturnController`）
- 订单评价与发货信息维护（`OrderReviewController` / `ShippingController`）

## 服务与训练
- 服务与服务分类维护（`ServiceController` / `ServiceCategoryController`）
- 服务预约管理（`ServiceAppointmentController`）
- 训练课程与预约管理（`TrainingCategoryController` / `TrainingCourseController` / `TrainingAppointmentController`）

## 公告与内容
- 系统公告与轮播图管理（`AnnouncementController` / `BannerController`）

## 搜索与统计
- 站内搜索（`SearchController`）
- 统计报表（`StatisticsController`）

## 文件与资源
- 文件上传/资源访问（`FileController`）
