# 运行清单

## 初始化数据库
- 在 MySQL 中创建数据库 `pet_home`
- 导入 `宠物之家/pet_home.sql`

## 后端启动
- 进入 `宠物之家/springboot`
- 按需更新 `src/main/resources/application.properties` 中的 `spring.datasource.*`
- 运行 `./mvnw spring-boot:run`（或 `mvn spring-boot:run`）
- 服务地址：`http://localhost:12345/`
- Swagger UI：`http://localhost:12345/swagger-ui.html`
- Knife4j Basic：`admin/admin`

## 前端启动
- 进入 `宠物之家/vue3`
- 安装依赖：`npm install`
- 启动开发环境：`npm run serve`
- 默认地址：`http://localhost:8080/`
- 前端代理：`/api` -> `http://localhost:12345`
